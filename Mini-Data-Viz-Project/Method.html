Data & Methodologies & Appendix

How Projects Completed

This section presents the final dataset and methodology used for plotting. To provide a clearer understanding of the visual methodologies employed in this project, each visualization tool used will be presented individually. This will assist the audience in comprehending the data and the insights gained from each visualization. The tools, which include choropleth maps, wordclouds, and pie charts, were selected for their ability to communicate the relationship between selected variables in a clear and concise manner. By presenting each tool separately, the audience will be able to appreciate the different aspects of the data that were revealed through the visualizations.

Data Set Overview

The final data set combines all JSON files and preprocesses the data to remove duplicates. Duplicates are identified by checking all columns, and any rows with identical values across all columns are removed.

Data Visualization Methodologies

Name Description Code
Figure 1: Bubble Chart This bubble chart, created using Tableau, displays the count of search parameters after data cleaning. The size of each bubble represents the corresponding count of the search parameter. These numbers can be used as a weight calculation in modeling. None
Figure 2: Choropleth Map The choropleth map is created using Plotly in Python, combining two maps into one. The first map displays the job count for each state, with colors representing the number of jobs available in each. The second map adds markers to the plot, indicating the rough location of the companies. Link
Figure 3: Word Cloud The WordCloud package in Python is used to create the word cloud. It combines all the job titles in the dataset, with each word's size representing its frequency. The word cloud is an effective way to display the most common job titles in the dataset. However, as the data is derived from search parameters (i.e., job titles), these parameters are added to the Stopwords list to remove them from the word cloud. Link
Figure 4: Bar Chart The bar chart is created using Plotly in Python, and it presents the top 10 platforms with the highest number of job postings. This graphical representation proves to be an effective method for showcasing the most prevalent platforms within the dataset. Link
Plot 5: Pie Chart A pie chart was created using Plotly in Python to show the frequency of job postings on the top three platforms. To generate the chart data, we first identified the top three platforms and then calculated the frequency of job postings for each of them. Link
