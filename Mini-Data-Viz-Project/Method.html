Methodologies & Reference

How Projects Completed

This section presents the final dataset and methodology used for plotting. To provide a clearer understanding of the visual methodologies employed in this project, each visualization tool used will be presented individually. This will assist the audience in comprehending the data and the insights gained from each visualization. The tools, which include choropleth maps, wordclouds, and pie charts, were selected for their ability to communicate the relationship between selected variables in a clear and concise manner. By presenting each tool separately, the audience will be able to appreciate the different aspects of the data that were revealed through the visualizations.

Data Set Overview

The final data set combines all JSON files and preprocesses the data to remove duplicates. Duplicates are identified by checking all columns, and any rows with identical values across all columns are removed.
